# Research Memory

## Focus

This vault is focused on understanding AI agents as a research and product domain. The goal is to build a durable, queryable knowledge base that tracks the core ideas, systems, tradeoffs, and companies shaping the space.

## Current Topic

- Primary theme: AI agents

## Core Questions

- What technically distinguishes an LLM application from an AI agent?
- Which memory architectures are actually useful in production agent systems?
- How do planning, tool use, and orchestration interact in modern agent stacks?
- What evaluation methods are reliable for measuring agent quality?
- Which frameworks, protocols, and products are becoming foundational infrastructure?
- Where are the strongest product and business opportunities in AI agents?

## Research Priorities

- agent architectures
- memory systems
- planning and reasoning strategies
- tool use and MCP-style interfaces
- orchestration frameworks
- evals, reliability, and safety
- startup and market implications

## What Good Output Looks Like

- concise concept pages that synthesize multiple sources
- map pages that explain how parts of the ecosystem fit together
- saved reports that answer concrete research questions
- explicit notes on contradictions, uncertainty, and open questions

## Working Preferences

- prefer synthesis over long raw summaries
- prefer durable markdown files over chat-only answers
- keep naming simple and readable in Obsidian
- preserve source links so claims can be traced back later
