# Anthropic

## Summary

Anthropic is a major organization in the agent ecosystem through its models, tool-use workflows, long-context positioning, and practical emphasis on coding and operator-style interactions. It is relevant here as both a model provider and a source of agent workflow patterns.

## Key Points

- Anthropic has influenced how many users think about agent loops, code execution, and filesystem-oriented workflows.
- Its product decisions are especially relevant to research on practical agent usability.
- Comparing Anthropic and OpenAI patterns is useful for separating durable architecture from vendor-specific design.
