# OpenAI

## Summary

OpenAI is one of the main organizations shaping the current AI-agent ecosystem through foundation models, tool-calling patterns, API design, and agent-oriented product primitives. It matters in this vault both as a model provider and as an influence on how agent workflows are packaged for developers.

## Key Points

- OpenAI has helped normalize tool calling, structured outputs, and agent-oriented API patterns.
- Its ecosystem matters for both research direction and product adoption.
- Changes in model capability can shift which agent scaffolds remain necessary.
