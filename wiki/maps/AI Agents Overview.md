# AI Agents Overview

## Summary

AI agents are systems that use language models to pursue goals through iterative reasoning, memory, tool use, and action. The category spans simple tool-using workflows through to multi-step systems with planning, execution loops, and external state.

## Key Points

- Agents differ from basic chat applications by acting over time rather than answering in a single turn.
- The core building blocks are usually planning, memory, tool use, orchestration, and evaluation.
- Most real systems involve tradeoffs between autonomy, reliability, cost, latency, and control.
- The strongest current opportunities are in constrained domains where tools, data, and feedback loops are well-defined.
